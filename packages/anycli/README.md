# anycli

`anycli` is an agent-friendly CLI for discovering and installing command-line tools by **slug**.

## Install

```bash
npm i -g @lightcity/anycli
```

## Usage

```bash
anycli search <slug> --json
anycli install <slug> --json
anycli install <slug> --yes --json
```

For full docs, see the AnyCLI website `/docs`.
